Full Stack Web Development Course

1. Introduction to Web Development

  • Web architecture overview: Frontend, Backend, Database
  • Understanding how websites work (client-server model)
  • Tools setup: Code editor (VS Code), terminal, Git, GitHub

2. Frontend Development

HTML & CSS
  • Semantic HTML5
  • CSS3 styling, Box Model, Flexbox, Grid
  • Responsive design & media queries
  • CSS preprocessors (SASS basics)
JavaScript (ES6+)
  • Syntax, DOM manipulation
  • Event handling
  • Fetch API, Promises, Async/Await
  • LocalStorage/sessionStorage
  • ES6 modules, arrow functions, destructuring
Frontend Framework – React.js
  • JSX, Components (functional/class)
  • Props, State, Lifecycle methods
  • React Hooks (useState, useEffect)
  • Forms, Validation
  • React Router
  • Context API or Redux (state management)

3. Version Control

  • Git basics: clone, commit, push, pull
  • Branching & merging
  • GitHub: Pull Requests, Issues, Collaboration

4. Backend Development

Node.js & Express.js
  • Introduction to Node.js
  • npm/yarn, package.json
  • Creating APIs with Express
  • Middleware, Routing
  • Environment variables, error handling
Authentication
  • User registration/login
  • JWT (JSON Web Tokens)
  • Sessions & cookies
  • OAuth (e.g., Google login)

5. Databases

MongoDB (NoSQL)
  • CRUD operations
  • Mongoose ODM
  • Data modeling, validation
  • MongoDB Atlas
(Optional) Relational Databases (MySQL/PostgreSQL)
  • SQL basics
  • Joins, indexes, normalization
  • ORM with Sequelize/Prisma

6. RESTful API Design

  • REST principles
  • HTTP methods & status codes
  • Building CRUD APIs
  • API testing with Postman

7. Deployment & DevOps Basics

  • Deploy frontend & backend: Vercel, Netlify, Render, or Heroku
  • CI/CD basics (GitHub Actions)
  • Environment configuration
  • Docker basics (optional)

8. Project Work

  • Mini projects throughout course
  • Final full-stack capstone project including:
    • User authentication
    • CRUD operations
    • Frontend + backend integration
    • Responsive design
    • Deployment & README

9. Bonus Topics (Optional/Advanced)

  • TypeScript
  • GraphQL basics
  • WebSockets (real-time apps)
  • Testing (Jest, React Testing Library, Supertest)
  • Progressive Web Apps (PWA)
  • Serverless functions (AWS Lambda, Netlify)
